# 桂糯龙—— 桂花糕主题宠物龙

> **属性**：甜点 · 自然系
>
> **性情**：温和、恋家、对湿度敏感
>
>
> **栖息地**：厨房、桂花树旁、蒸笼附近
>
>
> **代表花**：金桂 / 银桂
>
>
> **口头禅**：“糯～”
>
>

---


## 一、物种概述


桂糯龙是一种将传统中式糕点「桂花糕」与东方龙形结合而成的幻想生物。它的身体质感介于糯米糕与凝脂之间，散发着持久的桂花香气。


在自然状态下，它只有小型犬至大型犬的体型，但可以通过亲密度与特殊试炼进化到更优雅的形态。


---


## 二、完整进化树


```plain text
桂花凝露蛋 → 米果宝宝（幼年期） → 鸣桂幼龙（成长期） → 晶桂龙（成熟期）
                                              ↓ （限定进化）
                                          月宫桂神（究极体）
```

## 三、各阶段详解


### 阶段0：蛋 – 桂花凝露蛋


![Designer.png](images/blog/2026-04-01-36ddf5c0/img-ceb7e9f8cf.png)


| 项目       | 内容                                   |
| -------- | ------------------------------------ |
| **尺寸**   | 直径约12cm                              |
| **重量**   | 约320g                                |
| **外观**   | 半透明琥珀色蛋壳，内部悬浮着干桂花与微小光点。光照下可见糯米状核心蠕动。 |
| **触感**   | 温热、略有弹性，像刚出锅的水晶糕。                    |
| **气味**   | 若有若无的桂花蜜香，随孵化日期临近而变浓。                |
| **孵化台词** | _“咔…好甜的味道…是谁在叫我？”_                   |
| **背景音**  | 轻缓的蒸汽冒泡声（游戏内播放）。                     |

### 阶段1：幼年期 – 米果宝宝


![Designer-1.png](images/blog/2026-04-01-36ddf5c0/img-6ae4ab7936.png)


| 项目       | 内容                                          |
| -------- | ------------------------------------------- |
| **昵称**   | 糯团子                                         |
| **体型**   | 乒乓球大小                                       |
| **外观**   | 圆球形，表面沾满细白糯米粉；头顶微凹，有极小的肉翅（无法飞行）；眼睛绿豆大，黑色湿润。 |
| **移动方式** | 弹跳式滚动（像糯米糍掉在地上弹起）。                          |
| **性格**   | 胆小、黏人、害怕强风与干燥。                              |
| **食性**   | 只接受稀释的桂花蜜水（甜度≤20%），拒绝任何咸味或油脂。               |
| **特殊行为** | • 心情好时发出“糯～糯～”的哼唧声<br>• 被捧在手心会停止弹跳，安心睡觉     |
| **进化动画** | 身体拉长，弹出四肢与尾巴，米粉碎屑散落一地，发出“噗”的一声。             |

### 阶段2：成长期 – 鸣桂幼龙


![Designer-2.png](images/blog/2026-04-01-36ddf5c0/img-14150b3fcc.png)


| 项目       | 内容                                                                               |
| -------- | -------------------------------------------------------------------------------- |
| **体型**   | 家猫大小（体长约45cm，尾长30cm）                                                             |
| **体重**   | 约2.8kg                                                                           |
| **外观**   | • 乳白色短绒覆盖全身，质地像刚打好的糯米团
• 背脊有一排金黄色桂花鳞片（每片可散发真实桂香）
• 尾巴蓬松，横切面呈层层糕状
• 翅膀短小，仅能滑翔5~8米 |
| **瞳孔**   | 淡金色，细看有放射状纹路。                                                                    |
| **性格**   | 好奇心旺盛、活泼、喜欢藏食物（藏在尾巴里）。                                                           |
| **互动特点** | • 会主动把尾巴递给你闻（展示桂花香）
• 看到打开的蒸笼会兴奋地绕圈                                              |


---


### 阶段3：成熟期 – 晶桂龙


![Designer-3.png](images/blog/2026-04-01-36ddf5c0/img-12a054152a.png)


| 项目     | 内容                                                                                                        |
| ------ | --------------------------------------------------------------------------------------------------------- |
| **体型** | 大型犬（肩高65cm，体长1.2m）                                                                                        |
| **体重** | 约28kg                                                                                                     |
| **外观** | • 身体变为半透明的凝脂白色，内部可看到悬浮的金色桂花颗粒缓缓流动
• 犄角为淡黄色脆糖片，半透明
• 翅膀完全展开：由冷却后的糖浆结晶形成坚硬晶翅，飞行时洒下糖霜颗粒（纯视觉特效）
• 瞳孔像金桂花蕊的细丝状 |
| **性格** | 稳重、优雅、傲娇（对湿度有怪癖）：下雨天会主动出去淋雨，干燥天气会发脾气。                                                                     |
| **彩蛋** | 背上的桂花抽屉——打开鳞片暗格可存放3件小物品（如糖果、纸条）。                                                                          |
