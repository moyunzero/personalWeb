# **1. 一步解构和重命名**


在对象解构期间重命名变量，这在存在命名冲突时很有帮助。


```javascript
const user = { name: 'Alice', age: 25 };
const { name: userName, age: userAge } = user;
console.log(userName); // Alice
console.log(userAge);  // 25
```


# **2. 可选链式调用函数**


可选链可以与函数一起使用，确保在调用之前函数存在。


```javascript
const user = {
  getName: () => 'Alice',
};
console.log(user.getName?.());   // Alice
console.log(user.getAge?.());    // undefined
```


# **3. 使用** **`||=`** **运算符进行默认赋值**


逻辑或赋值（ `||=` ）仅在变量为 `null` 或 `undefined` 或类似 `0` 的假值时才赋值。


```javascript
let count;
count ||= 10;
console.log(count); // 10
```


# **4. 使用展开运算符将 NodeList 转换为数组**


展开运算符提供了一种快速将 `NodeList` 转换为数组的方法。


```javascript
const divs = document.querySelectorAll('div');
const divArray = [...divs];
console.log(Array.isArray(divArray)); // true
```


# **5. 使用默认值的数组/对象解构**


在解构时分配默认值以避免键缺失时出现 `undefined` 。


```javascript
const user = { name: 'Alice' };
const { name, age = 25 } = user;
console.log(age); // 25
```


# **6. 从数组中移除假值**


使用 `filter()` 删除数组中的 `falsy` 值（如 `0` 、 `null` 、 `undefined` 、 `false` ）。


```javascript
const arr = [0, 'hello', null, 42, false, 'world'];
const filtered = arr.filter(Boolean);
console.log(filtered); // ["hello", 42, "world"]
```


# **7. 按属性对对象数组进行排序**


轻松按特定属性对对象数组进行排序。


```javascript
const users = [{ name: 'Alice', age: 25 }, { name: 'Bob', age: 20 }];
users.sort((a, b) => a.age - b.age);
console.log(users); // [{ name: 'Bob', age: 20 }, { name: 'Alice', age: 25 }]
```


# **8. 动态导入以实现懒加载**


动态导入允许您仅在需要时加载模块，从而减少初始加载时间。


```javascript
const loadModule = async () => {
  const module = await import('./myModule.js');
  module.default(); // Calls the default export function
};
loadModule();
```


# **9. 默认参数与对象解构**


当使用默认参数时，您还可以解构并设置特定属性的默认值。


```javascript
function createUser({ name = 'Guest', age = 18 } = {}) {
  console.log(name, age);
}
createUser();               // Guest 18
createUser({ name: 'Alice' }); // Alice 18
```


# **10. 使用** **`Object.assign()`** **进行浅拷贝**


`Object.assign()` 方便在不更改原始对象的情况下进行浅拷贝。


```javascript
const original = { a: 1, b: 2 };
const copy = Object.assign({}, original);
copy.a = 3;
console.log(original.a); // 1 (unchanged)
```


# **11. 为性能优化缓存函数**


缓存基于参数的昂贵函数调用的结果，适用于计算密集型函数。


```javascript
const memoize = (fn) => {
  const cache = {};
  return (...args) => {
    const key = JSON.stringify(args);
    if (!cache[key]) {
      cache[key] = fn(...args);
    }
    return cache[key];
  };
};
const slowSquare = (n) => n * n;
const memoizedSquare = memoize(slowSquare);
console.log(memoizedSquare(4)); // 16 (cached on second call)
```


# **12. 使用** **`reduce`** **对数组项进行分组**


`reduce()` 可以根据属性对数组项进行分组，在数据处理中通常需要这样做。


```javascript
const people = [
  { name: 'Alice', role: 'admin' },
  { name: 'Bob', role: 'user' },
  { name: 'Charlie', role: 'admin' },
];
const grouped = people.reduce((acc, person) => {
  (acc[person.role] = acc[person.role] || []).push(person);
  return acc;
}, {});
console.log(grouped);
```


# **13. 使用** **`Array.flat(Infinity)`** **扁平化嵌套数组**


将多级嵌套数组展平变得简单易懂，使用 `Array.flat(Infinity)` 。


```javascript
const nested = [1, [2, [3, [4]]]];
console.log(nested.flat(Infinity)); // [1, 2, 3, 4]
```


# **14. 使用** **`!`** **切换布尔值**


切换布尔值就像应用两次 NOT 运算符一样简单。


```javascript
let isVisible = false;
isVisible = !isVisible;
console.log(isVisible); // true
```


# **15. 使用** **`concat()`** **合并多个数组**


`concat()` 有助于在单个语句中合并多个数组。


```javascript
const arr1 = [1, 2];
const arr2 = [3, 4];
const arr3 = [5, 6];
const merged = arr1.concat(arr2, arr3);
console.log(merged); // [1, 2, 3, 4, 5, 6]
```


# **16. 使用** **`for...of`** **和** **`await`** **进行异步数组迭代**


当遍历一个promises数组时， `for...of` 与 `await` 确保每个promise在下一个运行之前解决。


```javascript
const fetchData = async () => {
  const urls = ['url1', 'url2'];
  for (const url of urls) {
    const response = await fetch(url);
    console.log(await response.json());
  }
};
```


# **17. 快速获取数组中的最后一个元素**


获取数组中的最后一个元素，无需知道其长度。


```javascript
const arr = [1, 2, 3, 4];
console.log(arr.at(-1)); // 4
```


# **18. 使用** **`Intl`** **进行日期格式化**


`Intl.DateTimeFormat` 提供了一种跨地区格式化日期的强大方式。


```javascript
const date = new Date();
const formatted = new Intl.DateTimeFormat('en-GB', {
  dateStyle: 'full',
}).format(date);
console.log(formatted); // e.g., "Monday, 25 October 2021"
```


# **19. 使用** **`Math.round()`** **和模板字面量表示的整数**


模板字面量可以直接格式化圆整的数字。


```javascript
const num = 3.14159;
console.log(`${Math.round(num * 100) / 100}`); // 3.14
```


# **20. 使用** **`Array.from()`** **将类似数组的对象转换为数组**


```javascript
function example() {
  const argsArray = Array.from(arguments);
  console.log(argsArray);
}
example(1, 2, 3); // [1, 2, 3]
```
